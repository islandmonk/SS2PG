# SS_2_PG

A minimal Python application scaffold.

## Run locally

1. Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install the project:

```bash
pip install -e .
```

3. Run the application:

```bash
python ss_2_pg.py --name Alice
# or
ss2pg --name Alice
```
